# Sunamo NuGet Packages

Platform-independent .NET NuGet packages — shared libraries for all apps & devices.

## About

This is a collection of 120+ NuGet packages providing utility functions for:

- **String manipulation** — SunamoString, SunamoStringSplit, SunamoStringReplace, SunamoStringJoin, ...
- **Collections** — SunamoCollections, SunamoCollectionsGeneric, SunamoDictionary, ...
- **File I/O** — SunamoFileIO, SunamoFileSystem, SunamoGetFiles, SunamoPaths, ...
- **Async & Threading** — SunamoAsync, SunamoThread, SunamoThreading
- **Data formats** — SunamoJson, SunamoXml, SunamoYaml, SunamoCsv, SunamoHtml, ...
- **And many more...**

All packages target **.NET 8.0**, **.NET 9.0**, and **.NET 10.0**.

## Installation

Each package is available on [NuGet](https://www.nuget.org/profiles/sunamo). Install via:

```bash
dotnet add package SunamoString
```
